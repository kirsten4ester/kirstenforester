Tailored Shots


Tailored Shots is a client-to-service matching app designed to pair photographers with clients based on the client user's location, price point, photography needs and artistic preferences.

This site was built using Python and Django taking advantage of its built-in admin feature. This will conveniently allow the photographers to manipulate their data if they so choose to edit their price points or pictures that they would like to display.

The client users are also able to sign up for an account being required to have an account in order to enter the site. Once logged in, the client user will be able to filter through the data based on their set criteria. From there, they will then be able to save and favorite their search results and favorite photographers.

I was able to make this site mobile responsive with the implementation of Bootstrap.

In the process of developing and designing this site, I was given the opportunity to interview my client to learn their exact needs and vision in their web presence. I aimed to design a platform that served as a canvas to the many photographers using this platform to display their work. Furthermore, I implemented a clean and simple user interface to provide both client and service users a positive user experience. It was additionally of importance to deliver a product that is easily accessible both through desktop as well as mobile use when considering users will be greatly utilizing each medium.
