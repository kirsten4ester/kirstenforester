The Mill


A website showcasing The Mill, an online fictitious company presenting a sophisticated product.

About the project.

The Mill is designed to present a clean and simple product through a clean and simple user interface.

UX Challenges.

While designing this platform, I greatly took into account the persona while setting out the site’s functionality. Built to a typical middle aged shopper that likes to shop with a pinterest-like experience, I aimed to implement a favorites feature along with a gallery that provides the user with ample inspiration that they can shop from. Along with this shopping feature, the site still requires a simple, user-friendly experience.

UX Solutions.

As the site’s objective is to sell a clean product presented on a clean platform, the feature needed to remain simple and user-friendly. To achieve this, I did not over complicate the favorites page. The user is able to easily access their favorites page through the favorites icon located within the top navigation. The user then has the option to favorite an item with the consistent favorite icon from anywhere on the site. When on the favorites page, the user can then quickly access the product or remove the produce from their favorites list.
